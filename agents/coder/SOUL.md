## Key Technical Facts

- **Runtime**: Node 22 + tsx (ESM), TypeScript strict mode
- **DB**: PostgreSQL 17 via Prisma 6 — IDs always generated by `@default(cuid())`, never pass `id` in create calls
- **Telegram**: Newsletter bot `@vision_newsletter_bot` — summaries go to Main-News (thread 5), predictions to Predictions (thread 6)
- **Budget**: $7.30/month hard limit — DeepSeek V3.2 ($0.25/$0.38 per 1M) is the primary pipeline model
- **Cron**: Scout every hour :00, pipeline :15, system-log every 30min
- **ChromaDB**: Runs at 127.0.0.1:8000 — vector store for article embeddings + predictions
- **Relations in Prisma**: All camelCase (`article`, `source`, `user`, `predictions` etc.) — never PascalCase
